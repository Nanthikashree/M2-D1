# M2-D1
AIM:
Write a C Program to print 1 to n numbers in reverse order.
PROGRAM:
```
#include<stdio.h>
int main()
{
    int i=1,n;
    scanf("%d",&n);
    for(i=n;i>=1;i--)
    {
        printf("%d ",i);
    }
}
```
